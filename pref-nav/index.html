Pref-Nav: Preference-Aware Dual-Former MORL Policy for Social Navigation

Rashid Alyassi    Cesar Cadena    Robert Riener    Diego Paez-Granados
ETH Zurich
Paper PDF MORL UPE Sim
Awesome-List

Abstract

Social navigation enables mobile robots to operate in human-populated environments. However, existing social navigation policies often misalign with user expectations and lack mechanisms to incorporate post-deployment feedback for long-term human–robot interaction.

To address this, we propose Pref-Nav, a preference-aware multi-objective social navigation framework that adapts robot behavior in real time based on user feedback. Pref-Nav integrates a context-aware Graph-of-Thought LLM User Preference Encoder, which translates text-based preferences into objective weightings. These objective weightings condition a Dual-Former policy network trained with multi-objective reinforcement learning, enabling robots to align navigation behavior with diverse user needs and contexts. Experimental results demonstrate that Pref-Nav achieves strong consistency between expressed user preferences and the navigation behavior across varied scenarios.

Pref-Nav Architecture

We propose Pref-Nav, a Preference-Aware Dual-Former multi-objective policy for social navigation. In Pref-Nav, the robot's behavior is guided by a preference-conditioned, multi-objective learning policy.

The policy is trained using multi-objective reinforcement learning (MORL) to balance multiple, often competing objectives, including safety, efficiency, and comfort. At deployment, its behavior is dynamically adapted through a User Preference Encoder (UPE), which interprets text-based human feedback and translates it into a preference vector. This preference vector functions as a weighted sum of the objectives and adapts the policy in real time, allowing the robot to continuously align its navigation strategy with user intentions and context without retraining.
